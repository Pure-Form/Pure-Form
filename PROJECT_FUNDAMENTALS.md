# Project fundamentals — Pure Form

- Amaç: Kullanıcıların günlük kalori alımını takip etmelerini, kişisel fitness planları oluşturmalarını ve ilerlemelerini izlemelerini sağlayan uygulama.
- Diller: Türkçe ve İngilizce
- Öncelik: Android (ilk hedef)
- Teknoloji önerisi: Expo + React Native + TypeScript, Firebase backend (Auth, Firestore, Storage)
- Tasarım varlıkları: Stitch görselleri workspace içinde yüklü
- Yapay zeka entegrasyonu: ChatGPT/Gemini/Grok tarzı asistan modülü planlandı

İlerleme için bir sonraki adım: `npm install` çalıştırıp `expo start --android` ile scaffolding'i test et.
